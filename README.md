
# Violent


Violent is a high-performance, multi-purpose script menu for Roblox. Heavily inspired by the **Rampage Mod Menu** from GTA V, it focuses on a clean, keyboard-navigable UI, deep customization, and stability.

###  Quick Start
Paste this into your executor:
```lua
loadstring(game:HttpGet('https://vt-now.github.io/VIOLENT/vt.lua'))()
```

### Controls
*   **[F2]** - Toggle Menu Visibility
*   **[Arrows Up/Down]** - Navigate through options
*   **[Arrows Left/Right]** - Change slider values / Switch toggles
*   **[Return / Enter]** - Select / Confirm / Execute
*   **[Backspace]** - Go back to the previous tab
